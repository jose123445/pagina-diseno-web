El diseño elástico es una opción de diseño web que se presenta como una solución a las desventajas propias del diseño líquido y el diseño fijo. Una de las técnicas más utilizadas para hacer un diseño elástico es la de adaptar el tamaño de los objetos en la página al tamaño de letra seleccionada por cada usuario

En qué consiste el Diseño Web Elástico

Es una alternativa dentro del diseño web, que surge como solución a los inconvenientes que se presentan en los conocidos diseño web líquido y fijo. Uno de los procedimientos más usados para desarrollar un diseño web elástico, es el de llevar el volumen de los objetos en la página a las dimensiones de letra seleccionada por cada visitante. Día a día se suman más usuarios que viajan por la web mediante teléfonos celulares, tablets, o a través de una extensa gama de dispositivos; los cuales presentan diferentes resoluciones de pantallas, partiendo de 240 píxeles hasta los 1200 que poseen los PC convencionales de escritorio. La forma de convertir las webs ajustables a cada dispositivo es que la misma web revele desde cuál se está accediendo y a qué resolución, y se acondicione de acuerdo a esas especificaciones; es lo que se identifica en inglés como "responsive web design" que se traduce como "diseño de páginas web responsive", "elástico" o "flexible".
